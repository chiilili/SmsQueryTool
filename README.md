# SmsQueryTool 使用说明

## 安装方法

1. 打开 Google Chrome。
2. 在地址栏输入 `chrome://extensions/` 并回车。
3. 打开右上角的 `开发者模式`。
4. 点击 `加载已解压的扩展程序`。
5. 选择 SmsQueryTool 文件夹，也就是包含 `manifest.json` 的文件夹。
6. 确认扩展列表中出现 `SmsQueryTool`，并保持启用状态。


## 使用方法

1. 使用 Google Chrome 打开：

   `https://sms.jd.com/ext/TextSMSList.html`

2. 确认已经登录京东内网页面。如果提示鉴权失效，先打开：

   `https://kfuad.jd.com/`

   登录后刷新 SmsQueryTool 页面。

3. 在 SmsQueryTool 面板中点击 `获取数据`。

4. 等待数据读取完成。

5. 在 `人员` 下拉框中选择要查询的人员。

6. 点击 `开始`。

7. 查询过程中可以点击 `停止` 中断任务。

8. 查询完成后，如需导出命中结果，点击 `导出CSV`。

9. 如需重新查询，点击 `重置` 后重新执行以上步骤。
